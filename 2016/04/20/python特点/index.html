python特点

python 特点

  1. 简洁、代码少
  2. 解释型语言，边执行边翻译,依靠解释器跨平台
  3. 动态语言，弱类型语言，变量类型不定
  4. 代码无法加密，发布程序即发布源代码
  5. 应用场景：网站、后台、工具脚本、爬虫等

PS：那些大公司的代码不愿意开放的更重要的原因是代码写得太烂了，一旦开源，就没人敢用他们的产品了！
